C++ PROGRAMS

C++ is one of the thousands of programming languages that we use to develop software. C++ programming language is developed by Bjarne Stroustrup in 1980. It is similar to the C programming language but also includes some additional features such as exception handling, object-oriented programming, type checking, etc.

    Advantages
  1. C++ is a simple and portable structured programming language.
  2. It supports OOPs features such as Abstraction, Inheritance, Encapsulation.
  3. It provides high-level abstraction and useful for a low-level programming language, and more efficient for general-purpose.
  4. C++ is more compatible with the C language.
